# de-bar

---

一个de-bar工具，依赖于jq

简易控制台，开发、测试环境可以考虑启用。

---

## INSTALL


	fis install huya-fed/de-bar


## USAGE


	var Saidebar = require('sai-de-bar');
	Saidebar.log("log");
	Saidebar.info("info");
	Saidebar.warn("warn");
	Saidebar.error("error");


## API

### .log(String message)

### .info(String message)

### .warn(String message)

### .error(String message)

### .open()

### .close()
